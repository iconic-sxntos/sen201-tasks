# Student Course Registration System
This project is a Python-based Student Course Registration System developed as part of the SEN 201 assignment.
## System Overview
The system allows students to be registered and assigned courses using a simple console-based interface.
## SDLC Phases
The project follows the Software Development Life Cycle (SDLC):
- Requirement Analysis
- System Design
- Implementation
- Testing
- Deployment
- Maintenance
## Technologies Used
- Python 3
## How to Run
```bash
python student_course_registration_system.py
